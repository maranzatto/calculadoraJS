Sobre esta calculadora

Esta é uma calculadora básica com operações de soma, subtração, multiplicação e divisão. Foi criada como um projeto simples para praticar HTML, CSS e JavaScript vanilla.

O design é inspirado em interfaces modernas, com tema escuro e efeitos sutis de interação.

Funcionalidades:

  • Operações básicas (+ − × ÷)
  • Limpar tela (AC)
  • Apagar último dígito (⌫)
  • Suporte a vírgula decimal
  • Histórico da expressão no topo
  • Totalmente responsiva
